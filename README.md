# Instagram API Crawler

## Setup
```
pip install python-instagram
```

## Usage
https://instagram.com/developer/ から
* client_id
* client_secret
* access_token
を取得する.

参考サイト
http://www.dcrew.jp/ja-instagram-api-doc-v1/,
http://iaseteam.eshizuoka.jp/e1237677.html

取得したそれぞれを
search_insta_tag.py
に書き込む.

```
python search_insta_tag.py
```
で実行
